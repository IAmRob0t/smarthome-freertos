# J23 扩展板 V2 画板步骤（极简版）

目标：做一块只插 `J23` 的扩展板，包含：
- 1 个 `1x8` 孔座（对接主板 J23）
- 2 个 `1x4` 孔座（给两个模块）

你的指定连接规则：
- `3V3` 引出两路
- `GND` 引出两路
- `PA1` 只引到第一个 `1x4`
- `PC2` 只引到第二个 `1x4`

## 1. J23 引脚定义（按你给的图）

1. `J23.1 = 5V`（本版不用）
2. `J23.2 = 3V3`
3. `J23.3 = GND`
4. `J23.6 = PA1`
5. `J23.7 = PC2`

## 2. 两个 1x4 接口定义

建议统一丝印顺序（从 1 到 4）：
1. `VCC_3V3`
2. `GND`
3. `SIG`
4. `NC`

对应分配：
- `HDR_MQ135.3 = PA1`
- `HDR_LM393.3 = PC2`

## 3. 原理图连线

1. `J23.2 -> HDR_MQ135.1`
2. `J23.2 -> HDR_LM393.1`
3. `J23.3 -> HDR_MQ135.2`
4. `J23.3 -> HDR_LM393.2`
5. `J23.6 -> HDR_MQ135.3`
6. `J23.7 -> HDR_LM393.3`
7. `HDR_MQ135.4` 悬空（NC）
8. `HDR_LM393.4` 悬空（NC）

## 4. PCB 布局建议

1. 先固定 `J23 1x8` 位置和方向（1 脚标记必须清晰）
2. 两个 `1x4` 平行摆放，丝印清楚：`3V3 GND SIG NC`
3. `PA1`、`PC2` 分开走线，避免交叉
4. `3V3/GND` 线适当加粗（例如 20mil），信号线可 10mil

## 5. 上电前核对

1. `PA1` 没有接到第二个 `1x4`
2. `PC2` 没有接到第一个 `1x4`
3. 两个 `1x4` 的 `VCC` 都是 `3V3`，不是 `5V`
4. `J23` 方向和主板插接方向一致
